# Информационная система для доступа к многопользовательским базам данных с разным уровнем доступа
Этот проект создан для администрирования баз данных postgreSQL, расположенных на сервере.
Он позволяет запускать локальный веб-сервер, регистрироваться, авторизироваться, администрировать базы данных и управлять доступом к данным.

#Установка и запуск

Для установки сервера из исходных файлов, выполните следующие шаги:

1. Установите postgreSQL. Для установки перейдите на сайт https://www.postgresql.org/download/ и следуйте инструкциям;
2. Установите QtCreator 5.12. Для установки перейдите на сайт https://www.qt.io/download/ и следуйте инструкциям;
2. Скачайте проект с https://github.com/MAYER-Ti/WebServer/;
3. Скомпилируйте проект;
4. Поместите в полученную папку содержимое папки libs если ваша ОП Windows.

Для запуска сервера, выполните следующие шаги:

1. Запустите исполняемый файл WebServer.exe или WebServer.run из папки программы;
2. После запуска вы можете открыть браузер и перейти на адрес http://localhost:2323, где вместо localhost вы можете вписать ip устройства, запустившего сервер. Например -  192.168.0.111.

#Конфигурация
Вы можете настроить сервер, изменив значения переменных в файле etc/webserver.ini. Например, вы можете изменить порт, на котором работает сервер, или путь к каталогу с файлами.
